# API-and-Text-to-Image-generation-using-Gradio
Built a Text-to-Image generation system using deep learning and deployed it with an API using Gradio. The model converts text prompts into images in real time through an interactive interface, focusing on efficient inference, API integration, and user-friendly deployment.




### 📌 Project Overview

This project demonstrates a **Text-to-Image generation system** powered by deep learning and deployed using an interactive API built with Gradio. The application allows users to input text prompts and generate corresponding images in real time.

---

### 🎯 Objectives

* Build a **Text-to-Image generation pipeline**
* Develop an **API-based interface using Gradio**
* Enable real-time, user-friendly image generation from text prompts

---

### 🧠 Approach

* Implemented a deep learning-based generative model
* Designed an interactive UI using Gradio
* Created an API wrapper to handle text prompts and return generated images
* Optimized inference for quick response time

---

### 🛠️ Tech Stack

* Python
* Deep Learning
* Gradio


---

### 🚀 Features

* Text-to-image generation from natural language prompts
* Interactive web interface
* API-based architecture
* Easy deployment and sharing
